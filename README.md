# Buckley Workshop 1 — AI Essentials

`Buckley_Workshop_1.html` is a self-contained, single-file interactive presentation built for **The Buckley School's** first AI workshop session.

## What it covers

The presentation walks participants through the foundational concepts of modern AI in a scroll-based slide format, organized into the following sections:

- **The Foundation** — How machine learning works (training on examples, defining objectives, adjusting weights, shaping behavior)
- **The Landscape** — AI as a broad field, not just chatbots
- **Three Types of AI** — Predictive, generative, and agentic AI
- **AI Agents & Bots** — What agents are and how they differ from standard models
- **Vibe Coding** — A new approach to building software through natural language
- **Claude Code** — How to get started using Claude's desktop app without a terminal or IDE
- **The Players** — Overview of the major AI model providers and their flagship models
- **Local LLMs** — What it means to run a model locally and when that matters
- **Vertical LLMs** — Specialized models built for specific industries or use cases
- **The Anthropic Product Stack** — Claude's ecosystem of tools (API, Claude Code, Cowork, etc.)
- **The Three Claude Models** — Opus, Sonnet, and Haiku, and when to use each
- **Claude Mythos** — A deeper look at Claude's design philosophy
- **Workshop: Build Something Real** — A hands-on section where participants build **The Buckley School Smart Calendar**, a filterable, color-coded event calendar that works on mobile

## How to use it

Open `Buckley_Workshop_1.html` directly in any modern web browser — no server or dependencies required. Navigate by scrolling or using the top navigation bar. A slide counter in the bottom-right corner tracks progress.

## Intended audience

School faculty, staff, or students attending the Buckley AI workshop series. No prior technical background is assumed.
